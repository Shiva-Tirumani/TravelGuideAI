# TravelGuideAI+ (Enhanced)

## Overview
TravelGuideAI+ is an enhanced AI-based travel itinerary generator built using Streamlit and Google Gemini Pro.

## New Features
- Budget-based planning
- Travel type selection
- Interest-based customization
- Sidebar UI
- PDF export functionality
- Improved UI layout

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Add your Gemini API key in .env file

3. Run:
   streamlit run travel.py
